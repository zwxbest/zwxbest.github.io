    关于你走吧

    你走吧 是我的个人站。

    这个域名是我去年申请到的,靠手刷的。我觉得还不错,有意义,简短好记。比字母数字混杂,拼音缩写,非com后缀好多了。

    我像风一样自由

    就像你的温柔无法挽留

    你推开我伸出的双手

    你走吧最好别回头

    无尽的漂流自由的渴求

    所有沧桑独自承受

    关于写博客

    在自己的域名下,用自己的名字署名,有一种正式,庄严感,来监督自己写好每篇文章。

    每个人学习新东西,不论看视频还是看书,可能很快看完了,一个星期后也仅仅记得和目录一样的内容。我的看法是,再简单的部分也要用自己的思维和语言整理出来,代码再简单也要敲一遍。时间总是有的,很多人每天做地铁公交玩游戏,无聊刷朋友圈,用来丰富自己多好。

    同时也会记录程序员的生活。程序员并不是大家惯性思维中的木讷,长发胡渣那种。身边的很多朋友也有很多爱好-爬山,摄影,健身,说话超好听。我始终觉得写代码是一门艺术活,将写出优雅有美感的代码作为职业生涯的追求,虽然可能工作中没有多少机会发挥。
